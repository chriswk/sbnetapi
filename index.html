Welcome to sbnetapi.

Version 0.0.3

General Usage:

build.sbt / Build.scala

resolvers += "chriswk Artifactory" at "http://artifactory.chriswk.com/artifactory/libs-release-local"
libraryDependencies += "com.chriswk" % "sbnetapi" % "0.0.3"

Imports

import com.chriswk.bnet.wow.WoWApi

Features

Fetch realm list for region

WoWApi(region).findAllRealms().onComplete(req => {
  req.foreach(realms: List[Realm] => processRealmList(realms))
}))

Fetch Guild with region / name

Future[Guild] = WoWApi(region: String = "eu").findGuild(realm: String, name: String) // Use slug from Realm list

Fetch Guild with members

Future[Guild] = WoWApi().findGuild(realm, name, List("members"))

Fetch Character with region / name

Future[Character] = WoWApi(region: String = "eu").findCharacter(realm: String, name: String)

Data

Possible races

Future[Races] = WoWApi().races

Possible classes

Future[Classes] = WoWApi().classes

Guild data

Achievements
Future[Achievements] = WoWApi().guildAchievements
Perks
Future[Perks] = WoWApi().guildPerks
Rewards
Future[Rewards] = WoWApi().guildRewards
