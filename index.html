Welcome to sbnetapi.

Version 0.0.1

General Usage:

build.sbt / Build.scala

libraryDependencies += "com.chriswk" % "sbnetapi" % "0.0.1"

Imports

import com.chriswk.bnet.wow.WoWApi

Features

Fetch realm list for region

Future[List[Realm]] = WoWApi().findAllRealms()

Fetch Guild with region / name

Future[Guild] = WoWApi(region: String = "eu").findGuild(realm: String, name: String) // Use slug from Realm list

Fetch Character with region / name

Future[Character] = WoWApi(region: String = "eu").findCharacter(realm: String, name: String)
